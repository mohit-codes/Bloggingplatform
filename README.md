# Blogging ✍️ + QnA ❓ platform 
College Minor project using Spring boot, hibernate, Thymeleaf. 
A Blogging plus QnA system has been developed, to facilitate users’ growth and learning. The user
need to login or signup in order to access the system features. The Blogging module of the system
enables the user to post/add blogs/articles, read articles/blogs of other users, search blogs/articles by
category, manage previously posted blogs/articles, manage his/her profile details, like/upvote the
blogs/articles of other users, switch into QnA module. In the QnA module of the system users can
ask/add questions, answer the questions asked/added by other users, view the answers of other users
on a question, upvote the answers of other users, manage previously added/asked questions. The
admin module of the system can be accessed only by the admin and the admin can control the system
by managing users/blogs/articles/answers. The aim is to combine two separate modules into one web
based system so that users can benefit from it using only one account.
This Web based system is developed using spring boot, spring boot security, thymeleaf, hibernate,
bootstrap, MySQL.
